HOW IT WORKS

📤
01
IMAGE INGESTION
CT scan uploaded and preprocessed — resized, normalized, and optionally contrast-enhanced for optimal model input.
📐
02
SVM + PCA
100-component PCA reduces feature dimensionality. Linear SVM with probabilistic output performs binary detection: Hemorrhagic vs Normal.
🧬
03
VGG16 NETWORK
ImageNet-pretrained VGG16 with fine-tuned convolutional layers confirms detection. Ensemble consensus determines final verdict.
🗂️
04
SUBTYPE CLASSIFIER
Second VGG model classifies hemorrhage across all 4 taxonomy axes simultaneously. Per-category argmax extracts the best subtype prediction.
